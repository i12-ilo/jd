# 京东到家

## 项目功能

- 登录功能
- 首页展示
- 商户详情
- 订单管理

## 技术栈
**Vue3.0全家桶 + Axios + Sass**

## 难点
- 路由守卫进行登录校验
- 商品详情页与购物车数据的联动工能
- 使用watchEffect来实现代码拆分

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn serve
```

### Compiles and minifies for production
```
yarn build
```

### Lints and fixes files
```
yarn lint
```
